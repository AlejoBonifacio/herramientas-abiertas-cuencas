# Herramientas abiertas para cuencas hídricas

Este repositorio reúne **herramientas, protocolos y metodologías abiertas** orientadas al monitoreo y estudio de sistemas acuáticos y cuencas hídricas, con especial énfasis en enfoques socioecológicos y en contextos reales de trabajo de campo.

El objetivo es **documentar y compartir herramientas reproducibles**, desarrolladas o adaptadas en distintos proyectos, campañas y experiencias, para facilitar su uso, mejora y apropiación por parte de comunidades, instituciones educativas, equipos técnicos y científicos.

## ¿Qué tipo de contenidos hay en este repositorio?

- Protocolos de muestreo y medición ambiental.
- Metodologías de campo adaptadas a contextos con limitaciones logísticas.
- Herramientas de bajo costo y enfoques abiertos.
- Ajustes y aprendizajes derivados de experiencias reales de trabajo.

Este repositorio **no almacena datos crudos**, sino las herramientas necesarias para producirlos de manera transparente y reproducible.

## Estructura general

- `protocolos/`  
  Protocolos de medición y muestreo ambiental organizados por tipo de variable o método.

## Enfoque

Las herramientas aquí documentadas se conciben como **recursos abiertos**, sujetos a mejora continua y adaptación a distintos contextos territoriales y ambientales. No se presentan como recetas cerradas, sino como puntos de partida para procesos de monitoreo y aprendizaje compartido.
